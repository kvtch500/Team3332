## What This Session Shipped

**1. Apply to become a Captain (the stub is now real):**
- **Schema:** new `captain_applications` table (`user_id`, `motivation`, `experience`, `status` pending/approved/rejected, `created_at`, `reviewed_at`). Self-creates on boot via `schema.js` (CREATE IF NOT EXISTS) вЂ” no manual migration.
- **Backend (`routes/captain.js`):** `GET /apply/status` (member's latest application), `POST /apply` now **stores** the application (validates motivation, blocks a 2nd pending app with 409, blocks existing captains). DB is source of truth, not the token.
- **Backend (`routes/admin.js`):** `GET /captain-applications` (pending first), `PATCH /captain-applications/:id` approve/reject вЂ” **approving sets `users.is_captain = 1`**. `pending_applications` added to stats.
- **App (`app/index.html`):** new `CaptainApply` component replaces the old fake button вЂ” real form (motivation + experience), shows pending/approved/rejected states. Approved state tells the member to log out/in (token carries `is_captain`).
- **Admin (`admin/index.html`):** new **Captain Apps** page (approval queue + history) + gold "X applications awaiting review" banner on Overview.

**2. Member в†” Captain Q&A (the other half of the captain role):**
- **Schema:** new `captain_questions` table (`captain_id`, `member_id`, `question`, `answer`, `status` open/answered, timestamps).
- **Backend (`routes/captain.js`):** `GET /list` (captains a member can ask), `POST /questions` (member asks), `GET /questions/mine`, `GET /questions/inbox` (captain, open-first), `POST /questions/:id/answer` (only the owning captain).
- **App:** new **Ask a Captain** tab (members pick a captain, ask, see answers) + **Member Questions** inbox card inside the Captain Panel (reply inline).

**3. Capacitor native wrapper вЂ” scaffolded AND running on iOS simulator:**
- New `mobile/` folder: `capacitor.config.json` (appId `com.team3332.app`), `package.json` (Capacitor 6 + app/geolocation/splash/status-bar plugins), `sync-www.mjs` (copies `../app` в†’ `www/`, single source of truth), `SETUP-CAPACITOR.md` (full Mac guide), `ios-Info.plist-additions.xml` (background-GPS permission strings for later), `resources/README.md`.

**4. CORS fix (required for the native app to talk to the backend):**
